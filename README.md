# Salon Landing Page

Ein statisches Projekt für eine Salon Landing Page.

## Projekt Struktur

```
salon-landing-page/
├── assets/
│   ├── fonts/          # Schriftarten
│   ├── icons/          # Icons und SVGs
│   └── images/         # Bilder
├── css/
│   ├── base/           # Basis Styles
│   │   ├── reset.css
│   │   ├── variables.css
│   │   └── typography.css
│   ├── components/     # Wiederverwendbare Komponenten
│   │   ├── buttons.css
│   │   ├── cards.css
│   │   ├── forms.css
│   │   └── navigation.css
│   ├── layouts/        # Layout Styles
│   │   ├── header.css
│   │   ├── footer.css
│   │   └── grid.css
│   ├── utils/          # Utility Classes
│   │   ├── helpers.css
│   │   └── responsive.css
│   └── main.css        # Haupt CSS Datei
├── js/
│   ├── components/     # JavaScript Komponenten
│   │   └── navigation.js
│   ├── utils/          # Utility Functions
│   │   ├── helpers.js
│   │   └── scroll.js
│   └── main.js         # Haupt JavaScript Datei
├── index.html          # Haupt HTML Datei
├── .gitignore
└── README.md
```

## Technologien

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Verwendung

Öffne `index.html` in einem Browser, um die Landing Page anzuzeigen.
